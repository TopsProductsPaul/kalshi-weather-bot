# kalshi-weather-bot

Statistical arbitrage bot for Kalshi weather markets.

**See [SPEC.md](SPEC.md) for full specification.**

## Quick Start

```bash
# 1. Clone and setup
cp .env.example .env
# Add your Kalshi API credentials

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run (demo mode)
python main.py
```
